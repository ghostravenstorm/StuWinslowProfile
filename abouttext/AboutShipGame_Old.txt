Muffin Button - 2014
Deveolped in: JavaScript using Processing.js API
Development time: 8 hours
Team size: Self
Role: Programming

Simplistic graphical generation that renders a primitive face where it's eyes will track and follow the mouse cursor.

A program in Processing designed to be a 2D game with a very simple goal. The aesthetics are programmatically generated using Processing's graphic libraries, and the game logic for lose state, win state, movement, enemy behavior, and collision detection were all programmed from scratch.

# Instructions
- Press 'W' key to move toward your cursor.
- Press left-click to shoot towards your cursor.
- Press 'space' to reset the game.
