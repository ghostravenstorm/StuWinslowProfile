// Hotdog Flinger ------------------------------

Hotdog Flinger - March 2016 (Student/Intern Project) Developed in: Unity3D 5.2.1 Development time: 7 weeks Team size: 4 Role: Programmer Summary Game design project for Android that uses Google's Cardboard VR API to render a catapult for the user and Google's Android's voice recognition API to activate catapult actions. Technical Features > Designed and implemented a hands-free method of using voice activation to fire the catapult wrapping Google's API for voice recognition into Unity3D game scripting.
