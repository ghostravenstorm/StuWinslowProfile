// Portal Choas ------------------------------

Portal Chaos - Summer 2016 (Student/Intern Project) Developed in: Unreal 4.11 Development time: 3 months (WIP) Team size: 3 Role: Programmer and Designer Summary Game design research project where two users must work together to solve technical puzzles by sharing remote information that the other doesn't have individual access to. Technical Features > Uses a series of switches and buttons for user input. > Implements use of a Command design pattern process user input per generated puzzle configuration to determine if the submitted arrangement of input is correct for the current puzzle. > Designed a random puzzle generator framework (RPGF) using Blueprint interfaces in Unreal Engine 4 to create dynamic puzzles.
