// Nano Timers ------------------------------

Nano Timers - May 2017 (Personal Project) Developed in: Unity3D 5.6.1f1 Development time: 2 weeks (On-going development) Summary: High-precision timer API developed for Unity3D 5.X that can be integrated into multiple game-play systems. Technical Features > Uses a software based timer with 1/50th of a second precision. > Includes Stopwatches and Countdown Timers that do a function callback when a timer has expired to execute a specific event whether it be a countdown to game over, a repeating spawn timer for game entities, or a refresh timer for player abilities.
