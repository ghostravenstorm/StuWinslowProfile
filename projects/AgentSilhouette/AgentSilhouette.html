// Agent Silhouette ------------------------------

Agent Silhouette - Fall 2016 (Student Project) Developed in: Unity3D 5.4.0 Development time: 15 weeks Team size: 4 Role: Level Design, Sound Design, UI Design, and a very over-worked Programmer Summary: Agent Silhouette is a stealth game revolved around the player's ability to carefully time their jumps, rope swings, and to sneak past guards and other security devices without being detected.

Agent Silhouette also features a custom input controller I developed to handle random events that take button input such as switches and buttons that only toggle if these instances have reference to a valid input controller. This design approach more or less uses a Command pattern to process input.
