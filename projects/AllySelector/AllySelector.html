// AllySelector ------------------------------------

AllySelector - December 2016 (Personal Project) Developed in: Lua using Wildstar's Apollo API version 14 Development time: 4 weeks (On-going devvelopment) Summary: AllySelector is a UI addon and target selection assistant for Wildstar designed to help healers in PvE instances better keep track of their allies. Features: - Smart selection algorithm - Player bookmarking system - On screen styicky notes for bookmarks
