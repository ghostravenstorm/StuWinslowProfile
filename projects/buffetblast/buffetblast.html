// Buffet Blast ------------------------------

Buffet Blast - Summer 2017 (Student Project) Developed in: Unity3D 5.6.1f1 Development time: 7 weeks Team size: 6 Role: Sr. programmer Summary: Couch co-op free-for-all arcade game where 4 players must grab as much food as possible from the buffet counter and eat to explode before anyone else can!

Features a structured system I developed for keeping track of food types that I also designed with public interfaces in mind to be friendly to designers to make object accessibility simple.

The linked documentation is a sample of a more complete API of this game that only contains code I have contributed to this project.
